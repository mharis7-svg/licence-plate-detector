# License Plate Detector (YALPR)

## Description
This project detects and reads license plates from images and videos using **YOLOv8** and OpenCV.  
It supports processing datasets, extracting license plate regions, and reading the plates using OCR.

---

## Features
- Detect license plates in images or videos
- Extract license plate text
- Save processed outputs (images/videos)
- Supports dataset processing for training/testing
